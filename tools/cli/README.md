Cascode CLI

Command-line interface for the Cascode project. Distributed either as a .NET global tool (`Cascode.Cli`) or prebuilt binaries (see GitHub Releases).

Usage
- `cascode --help`
- `cascode synth examples/AmpAuto.cas -o build/AmpAuto.cir`
